Scrimish

My first major "project" was the final extra credit project of Stony Brook's CSE 114 course in Fall 2016: an implementation of the card game Scrimish. Written entirely in Java, it's a simple program that consists of a back-end for a simple console-based application as well as simple JavaFX-based user interface with text input. It was an interesting way to introduce me to creating programs that didn't focus on solely one concept taught by the course.

This really gave me inspiration to try and learn beyond the classroom. Source Code.
